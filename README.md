### Jawaban Tugas Teori

#### 1. Pengertian Class dan Object
* **Class:** Cetakan atau blueprint yang mendefinisikan atribut (data) dan method (perilaku) dari suatu entitas.
* **Object:** Realisasi atau perwujudan nyata dari sebuah class. Object memiliki identitas dan nilai yang spesifik.

#### 2. Jenis-jenis Method
* **Method Void:** Method yang hanya menjalankan perintah tanpa mengembalikan nilai (output).
* **Method Return:** Method yang mengembalikan nilai tertentu setelah proses selesai.
* **Setter (Mutator):** Method untuk mengisi atau mengubah nilai atribut.
* **Getter (Accessor):** Method untuk mengambil atau menampilkan nilai atribut.

#### 3. Analisis Kode Program (Gambar 1)
1. **Class Name:** Nama class adalah `Komputer`.
2. **Attributes:** Variabel `jenis_komputer` dan `merk`.
3. **Methods Block:** Area yang berisi kumpulan fungsi/perilaku object.
4. **Return Type:** Tipe data yang dikembalikan oleh method `getJenis()`.
5. **Getter Method:** Method `getMerk()` untuk mengambil data merk.
6. **Object Instantiation:** Pembuatan object baru `mykom` dari class `Komputer`.
7. **Method Call:** Pemanggilan method `setDataKomputer` untuk mengisi data.
8. **Output:** Perintah `System.out.println` untuk mencetak hasil ke layar.
